Studiums-Dashboard: B.Sc. Angewandte Künstliche Intelligenz

Über das Dashboard

Dieses Dashboard bietet einen umfassenden Überblick über deinen Studienfortschritt im B.Sc. Angewandte Künstliche Intelligenz.

Funktionen

  • Übersicht über alle absolvierten, laufenden und geplanten Module
  • Visualisierung des Studienfortschritts in ECTS-Punkten
  • Notenentwicklung und Durchschnittsberechnung
  • Modulverteilung nach Semestern
  • Detaillierte Tabelle aller Module mit Filterfunktionen

Technische Details

Das Dashboard wurde mit folgenden Technologien entwickelt:

  • Flask als Web-Framework
  • Dash und Plotly für interaktive Diagramme
  • SQLAlchemy und SQLite für die Datenspeicherung
  • Bootstrap für das responsive Design
